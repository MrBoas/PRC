Resumo

Este tpc tinha como objetivo converter um dataset de XML para RDF.

Para isso começei por criar a ontologia por detrás do dataset, usando o Protégé. Depois usei XSL para converter o dataset de XML para Turtle.

Após ter a ontologia e o dataset em Turtle, juntei-os para assim concluir a transformação em RDF.
