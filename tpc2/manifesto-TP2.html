Resumo

Este tpc pretendia converter um dataset de JSON para RDF. Para isso foi utilziado um conversor JSON para XML encontrado online.

Depois de ter o XML fui definir a ontologia por detrás do dataset, usando o Protégé. Depois criei um XSL para converter o dataset em XML para Turtle. Após ter isto feito, juntei ambos para concluir transformação em RDF.

Finalmente importei o ficheiro final obtido no GraphDB, com sucesso.
